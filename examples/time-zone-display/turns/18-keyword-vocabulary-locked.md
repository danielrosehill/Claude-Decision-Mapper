## Assistant

Locked.

**Three-keyword set for sourcing this product channel:**

1. **`industrial monitor`** — broadest, surfaces the right vendor universe (Beetronics, Faytech, Iiyama, Lilliput, AOpen, Elo).
2. **`open-frame monitor`** — best precision for the "no consumer chrome, VESA, HDMI in, DC barrel jack" form factor. Caveat: most "open-frame" still ships with a thin bezel; pure cabinet-embed naked-panel products are rarer.
3. **`embedded monitor`** — closely related; surfaces the genuinely cabinet/machine-mount-intended SKUs where the bezel is minimal or absent.

**Combine for best results:**
```
industrial open-frame monitor 15.6 HDMI
embedded monitor 15.6 VESA HDMI
open-frame industrial display 15 inch
```

**Keywords to avoid** (these surface the wrong category — display tablets, Android wall panels, smart signage):
- "wall mount tablet" / "meeting room tablet" / "PoE display"
- "Android display" / "smart display" / "signage tablet"
- "all-in-one panel" (ambiguous — sometimes returns the right thing, often returns Android tablets)

**Distinguishing signal in any listing:** if it mentions Android version / RAM / CPU / ROM / OS, it's a tablet. If it mentions HDMI input, DC voltage input, VESA, panel resolution, and nothing else — it's the right category.
